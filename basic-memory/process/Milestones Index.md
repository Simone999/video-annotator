# Milestones Index

This note is the durable router for milestone-tracking conventions. Use it when you need milestone meaning, status conventions, or archive location.

## Milestone Role

- milestones are transient roadmap grouping artifacts
- durable milestone conventions stay here and in `[[Schema Index]]`
- concrete milestone notes live under `archive/milestones/`

## Tracking Layout

```text
archive/milestones/
├── planned/
├── in_progress/
├── blocked/
└── done/
```
