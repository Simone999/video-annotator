# m-2: Review Workspace Completion

## Goal

Let a reviewer open one video and move through annotation work without raw frame-number or raw object-id entry for core review actions. This milestone is the current roadmap focus for operator review ergonomics.

## What To Implement

- useful default frame loading plus annotated-frame and keyframe navigation
- review controls for object context, timeline markers, and mask opacity
- core keyboard controls for review movement and annotation actions

## Checklist

- [ ] opening a video lands the reviewer on a useful frame without manual frame entry
- [ ] review navigation covers annotated frames, keyframes, and object context
- [ ] keyboard and mask-opacity controls are ready for routine review work
