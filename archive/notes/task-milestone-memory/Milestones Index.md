# Milestones Index

This note is the durable router for milestone-tracking conventions. Use it when you need milestone meaning, live Backlog usage, or legacy archive location.

## Milestone Role

- milestones are transient roadmap grouping artifacts
- durable milestone conventions stay here and in `[[Schema Index]]`
- live milestones now live in Backlog
- concrete archive milestone notes are historical or pre-Backlog snapshots

## Tracking Layout

```text
Backlog milestones:
- active until archived
- create them in implementation order when open roadmap grouping matters

Legacy snapshots:
- archive/milestones/planned/
- archive/milestones/in_progress/
- archive/milestones/blocked/
- archive/milestones/done/
```
