# Archive

Transient work lives here.

- `tasks/`: completed history and frozen pre-Backlog task snapshots
- `plans/`: active and historical plan notes
- `milestones/`: completed history and frozen pre-Backlog milestone snapshots
- `notes/`: historical audits, snapshots, and quarantined pre-calibration process notes
- `docs/`: retired repo docs snapshots

Live tasks and milestones now live in Backlog.
Quarantined pre-calibration task or milestone process notes live in `notes/temp-task-milestone-memory/` and are non-canonical.

`basic-memory/` is durable knowledge only.
